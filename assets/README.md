# Bot Avatar Assets

## bot-avatar.svg

The main bot avatar file. This is a high-quality vector image featuring:
- Deep blue gradient background (representing the heavens/Torah)
- Golden Star of David at the top
- Open book with "נ"ך יומי" (Nach Yomi) text
- Gold banner with "NACH YOMI" text

## How to Set the Bot Avatar

### Option 1: Convert SVG to PNG (Recommended)

1. Open `bot-avatar.svg` in a browser
2. Take a screenshot or use an online SVG-to-PNG converter
3. Recommended size: 512x512 pixels

Online converters:
- https://svgtopng.com
- https://cloudconvert.com/svg-to-png

### Option 2: Use BotFather

1. Open Telegram and message @BotFather
2. Send `/setuserpic`
3. Select your bot
4. Send the PNG image (512x512 recommended)

### Image Specifications for Telegram

- Format: PNG or JPG
- Recommended size: 512x512 pixels
- Maximum file size: 5MB
- The image should be square

Feel free to modify the SVG to:
- Change colors to match your brand
- Add your channel name
- Modify the text or styling

The SVG is fully editable in any vector graphics editor (Inkscape, Adobe Illustrator, Figma, etc.)
